# 🔍 ReversePrompt AI
 
## 📝 Descrição do Projeto
O ReversePrompt AI é um projeto de Engenharia Reversa Assistida por Inteligência Artificial desenvolvido utilizando Google AI Studio.

O objetivo principal do projeto foi analisar, compreender e reconstruir estruturas, comportamentos e fluxos de aplicações utilizando IA Generativa e técnicas de Engenharia de Prompt.

A aplicação explora como modelos de Inteligência Artificial podem auxiliar no processo de interpretação de sistemas, identificação de padrões e reconstrução lógica de funcionalidades através de comandos em linguagem natural.

Durante o desenvolvimento, foram utilizados recursos de automação inteligente, NLP (Processamento de Linguagem Natural) e Engenharia de Prompt para simular análises de engenharia reversa de maneira automatizada e assistida por IA.

---
