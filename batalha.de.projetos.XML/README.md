# 💻 Engenharia de Prompt e Aplicações em IA
 
## 📝 Descrição do Projeto
Este projeto consiste em um estudo prático de Engenharia de Prompt aplicado à geração automática de páginas web utilizando Inteligência Artificial.

A proposta do trabalho foi desenvolver um prompt estruturado em XML capaz de gerar uma página HTML5 single page com CSS integrado, contendo elementos modernos de interface, informações financeiras e recursos interativos.

O prompt foi testado em diferentes modelos de Inteligência Artificial, incluindo ChatGPT, Gemini, Claude, DeepSeek, Qwen, Grok e Maritaca, permitindo comparar desempenho, precisão, criatividade, quantidade de tokens e qualidade do código gerado.
