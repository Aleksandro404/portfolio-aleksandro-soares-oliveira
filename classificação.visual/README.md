# ⚽ Laboratório de Classificação Visual com IA
 
## 📝 Descrição do Projeto
Este projeto consiste em um experimento de classificação visual utilizando Inteligência Artificial através da plataforma Google Teachable Machine.

O objetivo principal foi treinar um modelo de IA capaz de distinguir visualmente pessoas classificadas como “Fã do Messi” e “Fã do Cristiano Ronaldo”, utilizando características visuais e padrões presentes nas imagens fornecidas durante o treinamento.

O projeto também teve como foco analisar como vieses podem surgir em sistemas de classificação visual, demonstrando como elementos como roupas, poses, expressões faciais e contexto visual podem influenciar diretamente os resultados gerados pela Inteligência Artificial.

Além do treinamento do modelo, foram realizadas análises críticas sobre impacto social, vieses algorítmicos e possíveis ações mitigadoras para tornar o sistema mais justo e equilibrado.

## 🔍 Análise do Viés

### ⚠️ Mecanismo do Viés

O modelo classificou pessoas como:

- “Fã do Messi”
- “Fã do Cristiano Ronaldo”

utilizando padrões visuais estereotipados presentes nas imagens de treinamento.

Isso demonstra como a IA pode aprender associações superficiais em vez de características realmente relevantes.

---

## 🌎 Consequência Social

Segundo a análise realizada:

- O impacto profissional é baixo
- O impacto emocional pode variar dependendo da pessoa
- A classificação pode reforçar estereótipos visuais

---

## 🛡️ Ação Mitigadora

Para reduzir o viés do modelo foram sugeridas ações como:

- Utilizar poses parecidas
- Padronizar expressões faciais
- Balancear cores de roupas
- Melhorar diversidade das imagens
- Aumentar quantidade de dados
