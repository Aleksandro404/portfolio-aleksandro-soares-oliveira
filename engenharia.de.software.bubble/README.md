# 🤖 Projeto Web com IA no Bubble
 
## 📝 Descrição do Projeto
Este projeto consiste em uma aplicação web desenvolvida na plataforma Bubble.io com auxílio de Inteligência Artificial e técnicas de Engenharia de Prompt.

O sistema foi criado utilizando recursos no-code para automatizar processos, organizar informações e desenvolver funcionalidades web de forma rápida e intuitiva, sem necessidade de programação tradicional.

A aplicação demonstra como ferramentas de IA podem auxiliar no desenvolvimento de interfaces, workflows e estruturação de sistemas completos utilizando Bubble.io.

Além do desenvolvimento da aplicação, o projeto também aborda estratégias de exportação de dados, integração via API e possibilidades de migração futura para outras tecnologias.

---

## 🚀 Tecnologias Utilizadas

- Bubble.io
- Inteligência Artificial
- Engenharia de Prompt
- No-Code Development
