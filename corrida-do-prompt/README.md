# 🎨 Projeto de Engenharia de Prompt com IA Generativa
 
## 📝 Descrição do Projeto
Este projeto consiste em um estudo prático sobre Engenharia de Prompt utilizando Inteligência Artificial Generativa para criação e refinamento de imagens.

O trabalho foi desenvolvido a partir da criação de uma imagem base de um astronauta em estilo barroco tocando violoncelo em Marte, realizando múltiplas iterações de prompts para analisar como pequenas alterações nas instruções impactam diretamente o resultado gerado pela IA.

Durante o desenvolvimento, foram aplicados conceitos da Taxonomia de Bloom e Taxonomia de Fink para avaliar o processo de aprendizagem, criatividade e interpretação da Inteligência Artificial.

O objetivo principal foi compreender como descrições detalhadas, contexto visual e pequenas mudanças semânticas influenciam a geração de imagens por modelos de IA.

## 📊 Resultados e Aprendizados

O projeto apresentou resultados positivos na análise da interação entre usuário e IA.

- Refinamento progressivo dos prompts
- Aprendizado sobre interpretação contextual da IA
- Entendimento da importância de detalhes específicos
- Aplicação prática da Taxonomia de Bloom
- Aplicação prática da Taxonomia de Fink
- Evolução visual da imagem através de iterações

## 🧠 Processo de Desenvolvimento

### 🎯 Prompt Inicial

> “Uma imagem de um astronauta estilo barroco tocando violoncelo em marte”

A partir desse prompt inicial, novos elementos foram adicionados progressivamente para analisar como a IA interpretava contexto, posição dos objetos, ambiente e detalhes visuais.

---

## 📚 Relação entre os Arquivos

### 📄 Carta do Pirata (PDF)

O arquivo contém os diferentes prompts utilizados durante o experimento, organizados em etapas progressivas.

Cada prompt adiciona novos detalhes na imagem para observar:

- Mudanças visuais
- Precisão da IA
- Interpretação contextual
- Impacto de pequenas alterações semânticas

---

### 📄 Trabalho IA (DOCX)

O documento apresenta:

- Análises utilizando Taxonomia de Bloom
- Reflexões utilizando Taxonomia de Fink
- Avaliação crítica dos resultados gerados
- Discussão sobre criatividade e refinamento dos prompts

---

## 🔍 Principais Observações

### ✅ Taxonomia de Bloom

Foi utilizada para analisar:

- O que faltava nos prompts
- Se o tom visual estava correto
- Se a IA ignorava elementos importantes
- Se o grupo estava inovando ou apenas repetindo padrões

---

### ✅ Taxonomia de Fink

Foi utilizada para compreender:

- Como mudanças pequenas alteram o resultado
- Como contexto geográfico influencia a IA
- Como detalhes específicos aumentam precisão visual

Exemplo observado:

> Quando não foi especificado corretamente onde a bandeira deveria aparecer, a IA criou novas pedras próximas ao astronauta para posicioná-la.
